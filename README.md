### What I'm building

**[Spectres](https://spectres.pro)** — a professional crypto community platform.

I built Spectres because most crypto chat tools are chaotic. Telegram channels get flooded with images and spam. Discord is overwhelming. Twitter's algorithm decides what you see. There was no clean, focused place for real crypto discussion.

Spectres fixes that:

| Feature | What it does |
|---|---|
| **Public Rooms** | Create a room, share the link, anyone can join and chat |
| **Analyst Channels** | Owner-only broadcast — you post, subscribers read |
| **Live Prices** | 12 coins via CoinGecko, real-time updates |
| **Trading Tools** | P&L, position size, risk/reward, liquidation price, coin converter |
| **Text only** | No images, no media, Free — forever |

**Powered By:** CoinGecko
